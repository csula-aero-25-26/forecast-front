## Getting Started

1. Clone the repo:
```
git clone https://github.com/csula-aero-25-26/forecast-front
```

2. Go to the project's root folder and use npm to install all required components:
```
npm install
```

3. Launch the project in developer mode:
```
npm run dev
```
